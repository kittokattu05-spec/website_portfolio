# Keith's Portfolio Website

A retro-inspired, minimalist portfolio website with OS-like interface and draggable windows.

## Features

- Boot sequence animation
- Desktop interface with icons
- Draggable windows for different sections
- Light/Dark theme toggle
- Responsive design

## How to Run

1. Start your local server (e.g., XAMPP Apache)
2. Open `index.html` in your browser

Or simply open `index.html` directly in a modern browser.

## Structure

- `index.html` - Main HTML file
- `css/style.css` - Stylesheet
- `js/script.js` - JavaScript functionality

## Sections

- Home (index.html)
- About (readme.txt)
- Projects (projects/)
- FAQ (faq.txt)
- Links (links.url)
- Contact (contact.info)
